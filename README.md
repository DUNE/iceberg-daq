# iceberg-daq

Scripts and utilities used for configuring and running DUNE DAQ in ICEBERG. 

The docoumentation on how to use the scripts can be found in the [ICEBERG DAQ wiki](https://wiki.dunescience.org/wiki/ICEBERG_DAQ).
